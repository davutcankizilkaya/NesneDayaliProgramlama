# 🏦 Nesneye Dayalı Programlama - Banka Hesabı Projeleri

Bu repo, Nesneye Dayalı Programlama (OOP) dersinde geliştirdiğim banka hesabı simülasyonlarını,kantin restoran similasyonlarını içermektedir.

## 📌 Proje İçeriği

Bu projede aşağıdaki kavramlar uygulanmıştır:

- Sınıflar (Classes)
- Nesneler (Objects)
- Encapsulation (Kapsülleme)
- Kalıtım (Inheritance)
- Polimorfizm (Polymorphism)

## 💻 Projeler

### 1. Banka Hesabı Sistemi
- Hesap oluşturma
- Para yatırma
- Para çekme
- Bakiye görüntüleme

### 2. Gelişmiş Banka Sistemi
- Farklı hesap türleri (Vadeli, Vadesiz)
- Faiz hesaplama
- Kullanıcı yönetimi

🍽️ 3. Kantin / Restoran Sistemi
Masa oluşturma
Ürün ekleme (yiyecek/içecek)
Sipariş alma
Masa hesabı hesaplama
Adisyon (fiş) oluşturma
